# Workspace Structure

Root
- README.md - Short project title header.
- requirements.txt - Python dependencies for extraction, filtering, and training.
- 2504.16891v1.pdf - Reference method paper PDF.
- .gitignore - Git ignore rules.
- .git/ - Git metadata directory.
- src/ - Python package for scraping and data pipeline.
  - __init__.py - Package marker.
  - cot_synthesis/
    - __init__.py - Placeholder package for chain-of-thought synthesis.
  - data_pipeline/
    - __init__.py - Package marker.
    - extract.py - PDF text extraction to JSONL with rule-based parsing.
    - filter_rules.py - Rule-based filtering (multiple choice, image-required, language).
    - filter_validity.py - LLM-based validity filter using vLLM.
    - to_csv.py - Normalize and export JSONL to CSV or JSONL.
    - dedup.py - Embedding-based deduplication and decontamination.
  - scraping/
    - __init__.py - Package marker.
    - scrape_defantri.py - Scrape index and download OSN PDFs.
  - training/
    - __init__.py - Placeholder package for training scripts.
- notebooks/
  - filter_validity_kaggle.ipynb - Notebook for validity filtering experiments on Kaggle.
- data/
  - raw/
    - osn/
      - sample: unknown_Download_File_1-MDRMez.pdf

      - .....
